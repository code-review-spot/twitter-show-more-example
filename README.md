# Show More View

Backbone app to show more tweets

## Getting Started
clone this repository

rename config.sample.js to config.js

populate config.js with your twitter API credentials

```npm install -g grunt```

```npm install```

```grunt server```
 
*Live App
http://localhost:1234

*Unit tests
http://localhost:1234/tests/show-more-view.html
